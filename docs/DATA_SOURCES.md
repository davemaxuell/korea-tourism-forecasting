# Data Sources and Reuse Notes

This repository contains small raw and processed data files so the analysis can be reproduced from the public project files. Reuse rights may differ by source. Check each provider's current terms before redistributing derived datasets outside this project.

## Foreign Visitor Arrivals

- Source family: Korea Tourism DataLab / Korea Tourism Organization tourism statistics.
- Project use: monthly foreign arrivals for tourism purpose.
- Official site: https://datalab.visitkorea.or.kr/
- Notes: The public pipeline filters the tourism-purpose row from the raw Excel files and normalizes each observation to month-start dates.

## K-Culture Search Interest

- Source family: Google Trends exports for Web Search and YouTube Search.
- Project use: monthly search-interest index signals for K-pop, K-drama, K-food, K-culture, and related keywords.
- Google Trends: https://trends.google.com/trends/
- Google Trends data FAQ: https://support.google.com/trends/answer/4365533
- Notes: Google Trends values are normalized indices, not absolute search counts. This project therefore uses `*_search_index` naming.

## Exchange Rates

- Source family: Bank of Korea ECOS.
- Project use: monthly KRW-denominated exchange rates where coverage is sufficient.
- ECOS API portal: https://ecos.bok.or.kr/api/#/
- Notes: The clean baseline uses `usd_krw` and `jpy_krw_100`. `cny_krw` is excluded from the baseline dataset when coverage is too sparse.

## Generated Data

Processed CSV files under `data/processed/` are generated by:

```bash
python src/build_dataset.py
```

Generated model outputs under `outputs/` are intentionally ignored by Git. A compact result summary is committed under `reports/`.
